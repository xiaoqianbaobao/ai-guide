# AI 基础知识

人工智能是计算机科学的一个分支，致力于创造能够执行通常需要人类智能的任务的机器和软件系统。

## 什么是人工智能？

人工智能（AI）是指计算机科学中模拟、延伸和扩展人类智能的理论、方法、技术及应用系统。它试图创造能够感知环境、获取知识、进行推理、学习、规划、解决问题和与人类交互的机器。

## AI 的主要分支

### 1. 机器学习（Machine Learning）
机器学习是 AI 的核心子领域，专注于让计算机从数据中自动学习，而无需明确编程来实现特定任务。

### 2. 深度学习（Deep Learning）
深度学习是机器学习的子领域，使用多层神经网络来模拟人脑的学习过程。

### 3. 自然语言处理（NLP）
自然语言处理致力于让计算机理解和生成人类语言。

### 4. 计算机视觉（Computer Vision）
计算机视觉使机器能够解释和理解视觉世界。
