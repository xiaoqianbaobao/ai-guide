# 欢迎访问 AI Guide

AI Guide 是一个专注于人工智能工程实践的中文知识体系，致力于为开发者提供系统、实战的 AI 学习资源。

## 📚 内容导航

- **[基础层](/guide/)** - AI 核心原理与算法基础
- **[模型层](/models/)** - 大模型能力解析与选型
- **[工程层](/engineering/)** - AI 系统架构与部署实践
- **[Agent 层](/agent/)** - 智能体系统与应用开发 ⭐
- **[应用层](/application/)** - AI 产品化与商业化方法论
- **[资源层](/resources/)** - 持续学习与工具推荐

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📖 内容特色

- **系统性**：从基础到高级，构建完整的知识体系
- **实践性**：每个知识点都配有可运行的代码示例
- **前沿性**：紧跟 AI 领域最新发展，及时更新内容
- **中文友好**：深度解析中文术语，降低学习门槛
